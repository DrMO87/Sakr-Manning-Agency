🔐 Permission Roles Summary

Role Description Access Level
Admin System Administrator Full access to all resources
HR Manager Human Resources Manager Manage all resources except Admin operations
Recruiter Recruitment Specialist View most resources, edit candidates and CVs
Employee Regular Employee View/edit own profile and submissions only
