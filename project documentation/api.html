Project Overview

Sakr Manning Agency API is a crew management system designed for maritime operations.

Core Features

  • Seafarer Profile Management: Complete personnel records with documents, certificates, and health data
  • Rank & Certification Tracking: Automated rank code generation and certificate assignments
  • Contract Management: Employment history with vessel assignments
  • Sea Service Records: Historical voyage data
  • Document Management: Passport, seaman's book, and health certificates
  • RESTful API: Full CRUD operations with nested serialization
Tech Stack: Django 4+, Django REST Framework, django-filter, PostgreSQL (implied)

Installation Steps

# Clone repository
git clone <repository-url>
cd sakr-manning-api

# Create virtual environment
python -m venv venv
source venv/bin/activate  # Linux/Mac
# or
venv\Scripts\activate  # Windows

# Install dependencies
pip install -r requirements.txt

# Run migrations
python manage.py makemigrations
python manage.py migrate

# Create superuser
python manage.py createsuperuser

# Run development server
python manage.py runserver

Key Features

Auto-Generated Username

When creating users via API:

  • Username auto-generated from email prefix
  • Ensures uniqueness by appending numbers: john.doe, john.doe1, etc.

Automatic Rank Code Assignment

UserRank model auto-generates sequential codes:

  • Pattern: {prefix}.{sequence}
  • Example: DO-1.001, DO-1.002, ER-4.001

Nested Serialization

  • Read: Full nested objects (ranks, certificates, etc.)
  • Write: Accepts ID lists (rank_ids, certificate_ids)

Usage Examples

Create User with Certificates & Ranks

POST /api/users/create/
Content-Type: multipart/form-data

email: john.doe@example.com
first_name: John
last_name: Doe
rank_ids: 1,3,5
certificate_ids: 2,4,6
profile_image: <upload file>

Add Rank to User

POST /api/users/5/ranks/add/
Content-Type: application/json

{
  "rank_id": 3
}

Response

{
  "message": "Rank '2nd. Engineer' added to user John Doe",
  "user_rank": {
    "id": 10,
    "assigned_code": "EO-2.015",
    "rank_code": "EO-2.000",
    "rank_name": "2nd. Engineer",
    "rank": 3
  }
}
