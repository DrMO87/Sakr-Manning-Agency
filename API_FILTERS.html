Introduction

Welcome to the Sakr Agency Filter API Documentation. This guide details how to filter resources across our system using query parameters.

Base URL: http://api.sakr.com/api/

All endpoints support query strings, e.g., ?status=Active&name=John.

1. Core API (User & General)

Users & Crew

GET /users/
ParameterDescription
namePartial match (e.g., "John")
rank_nameRank title (e.g., "Master", "Cook")
nationalityCountry name
user_statusCurrent status (On Site, Vacation)
is_blacklistedBoolean (true/false)
Example: Find available Ukrainian Masters.
?nationality=Ukraine&rank_name=Master&user_status=On Site

Finance Records

GET /finance-records/
ParameterDescription
companyCompany ID
statusPayment status (Pending, Paid)
start_date_fromStart date range (YYYY-MM-DD)
Example: Pending payments for December.
?status=Pending&start_date_from=2024-12-01

Interviews

GET /interviews/
ParameterDescription
candidateUser ID
statusScheduled, Completed, Rejected
scheduled_dateSpecific date (YYYY-MM-DD)

CV Submissions

GET /cv-submissions/
ParameterDescription
positionPosition ID
statusApplication status (Pending, Approved)

2. Operations & Companies

Companies

GET /companies/
ParameterDescription
nameCompany Name
company_typeType (e.g., "Ship Owner")
statusActive, Inactive

Job Orders

GET /companies/job-orders/
ParameterDescription
statusOpen, Closed, Hold
reference_numberJO Reference Code
request_date_fromDate range start
Example: Find open Job Orders from a specific ship.
?status=Open&ship=5

3. Fleet & Ships

Ships

GET /ships/
ParameterDescription
nameVessel Name
imo_numberIMO Number
flagFlag State (e.g., Panama)
ship_typeVessel Type (e.g., Tanker)
statusActive, Inactive
Example: Active Tankers under Panama flag.
?flag=Panama&ship_type=Tanker&status=Active
