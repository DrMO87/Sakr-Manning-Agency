2. DATA FLOWS

User Data Flow (Seafarer)

  1. Registration: User registers via /api/users/register/.
  2. Authentication: User logs in to receive JWT token.
  3. Profile: User completes profile (Personal info, Certificates) via /api/users/users/{id}/.
  4. Documents: User uploads documents via /ai/upload/ for AI processing.
  5. Application: User submits CV for jobs via /api/users/cv-submissions/.

Admin Data Flow

  1. User Management: Admin views/filters users via /api/users/users/.
  2. Master Data: Admin manages Companies, Ships, Ranks via respective endpoints.
  3. Recruitment: Admin schedules interviews and reviews CVs.
  4. Logistics: Admin manages Tickets and Traveling Papers.
  5. Automation: Admin batch converts AI-processed applicants to users.

8. AUTHENTICATION HEADERS

For protected endpoints, include the authentication token in the request header:

Authorization: Token <your_token_here>

Or if using JWT:

Authorization: Bearer <your_jwt_token_here>
